\# Mega Unit Conversion



A simple Python console-based application that performs multiple unit conversions in a single program.



\## Features

\- Length conversions (meter, kilometer, centimeter)

\- Weight conversions (gram, kilogram, pound)

\- Temperature conversions (Celsius, Fahrenheit, Kelvin)

\- Stores and displays conversion history

\- Modular Python code

\- Beginner-friendly design



\## How to Run

```bash

python main.py





How to Run Tests
